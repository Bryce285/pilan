# OfflinePiFS - Roadmap

## Phase 0
* Use Buildroot toolchain to boot a custom linux image on Pi5
* Get serial console output

## Phase 1
* Custom init
* Network auto-config
* Service auto-start
* No shell exposed

## Phase 2
* Implement file exchange MVP
* Upload/download
* Local storage
* Basic encryption

## Phase 3
* Add robusticity
* Power loss testing
* Corrupt file handling
* Watchdog reset
* Logging
